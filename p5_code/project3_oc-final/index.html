Project 3 - Personal Clock

(Artist Sketch) In this project, I created a personal 24-hour clock that represents my activities throughout the day. The number of images in the top (two-row) grid represents the seconds. The number of images in the middle (three-row) grid represents the minutes. The number of images in the bottom (two-row) grid represents the hour.
