Exercise 5 - PHASE 1

(Artist Sketch) This sketch is Phase 1 of Exercise 5.
In this sketch, I created 2D array of numerical values and a 2D array of string (text) values to represent Pokemon characters. The original mapping of the numerical array is the to the greyscale grid of Eevee (the deer-like pokemon). The original mapping of the text array is to the grid of color text for Charmander (the orange pokemon). This sketch also includes each 2D array remapped two more times in distinct ways using different colors, shapes, and bitmaps.
