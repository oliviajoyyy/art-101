Exercise 6

(Artist Sketch) There are 5 scenes in this exercise, which can be nagivated through key bindings or by use of the gui buttons on the screen. This exercise uses the libraries p5.scenemanager, p5.sound, and p5.toughgui.
Key bindings:
'1' - intro
'2' or 'h' or 'H' - scene help
'3' - alien scene 1
'4' - alien scene 2
'5' or 'e' - end scene
